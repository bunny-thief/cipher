CIPHER n

Cipher n

Usage: A message to be encrypted is passed by entering it into the text area below. The shift amount is included at the end of the message and needs to be separated by a space. The shift amount can be positive or negative.

After the text is entered, clicking on the submit button will take the text and apply the shift, remove the shift number from the end of the input text and return the encrypted text below the text area.

The input text area and encoded output message can be cleared by clicking on the Clear button.

Entering the following text into the text area We attack at dawn 15 would return Lt piiprz pi splc. To decode the message you simply paste the encoded message and add -15 at the end: Lt piiprz pi splc -15.
